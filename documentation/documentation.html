King UI was created by designers for designers, King UI / Components features over 350+ UI Elements, Sections and Components, all coded in HTML5, CSS3 with SASS and based on the latest Bootstrap version. King UI is huge time saver, a must have UI / Components Kit for every web / UI designer or web developer.

Start your Project

You can easily create your web project using King UI - Bootstrap UI & Components Kit.

Go to King UI main folder and open index.html file or just copy the html content below and just replace <!-- your content here --> with your html or you can see the how to use guide here , in order to see how to use the UI Elements & Components. Please note: All CSS and JS files from demo are included, so unwanted elements can be removed.

index.html content:

Setup Contact Form

You can setup the contact form to send emails to your email address in php / form-process.php file edit this code from line 35:

          $EmailTo = "hello@example.com";

Replace the dummy email address hello@example.com with your own email address

In form-process.php you can edit the subject of the emails you will get by editing this code, from line 36:

          $Subject = "New Message Received";

Replace New Message Received from the above code with the subject you want to see when you receive an email from this contact form
